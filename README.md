# Página web de E-Valuar

E-Valuar es una empresa de consultoría dedicada al análisis y evaluación de políticas públicas con más de 15 años de experiencia en Colombia y la región, en este repositorio se almacena la información de la página web, la cual fue desarrollada en R y Shiny utilizando como referencia los desarrollos de [Hugo Apéro](https://hugo-apero-docs.netlify.app/).
